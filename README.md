# Teiosteppa Tap

## How do I install these formulae?

`brew install teiosteppa/tap/<formula>`

Or `brew tap teiosteppa/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "teiosteppa/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
